Overview

Password Maker is a simple tool that will create random passwords and then encrypt the chosen password using one of the four symmetric key algorithms available in .NET. It allows you to toggle the byte representation between decimal, hexidecimal and Base64 encoding formats.

The private symmetric key and initialization vector will be automatically generated.

You can also paste a password of your choice into the password field.

Random Passwords

Choose a password length and then click the Random button to generate a random password. Under the Random menu are options to include or exclude various characters from the radomizer. Unchecking Similar Characters will remove characters like upper case O and lower case l from the set of characters to chose from.

Encrypting

Click the Encrypt button to have the string in the password field encrypted using a symmetic key and init vector. The key and init vector will be chosen randomly each time you click Encrypt.

The default symmetric key algorithm is AES/Rijndael using the largest key size. You can change the algorithm and key size under the appropriate menus.

Copying

Finally, you can copy the encrypted version of the password, the private key and the init vector to use elsewhere. The Delimiter menu toggles what separates each byte value in the decimal and hexidecimal format.
